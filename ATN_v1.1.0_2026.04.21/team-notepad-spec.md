# 팀 메모장 (Team Notepad) — 프로젝트 스펙

## 프로젝트 개요

Anchors(앵커스) 브랜드 톤앤매너를 적용한 팀/업무용 메모장 웹 앱.  
심플하고 깔끔한 UX를 기반으로, 딥 다크 테마와 Bold한 디지털 감성을 반영한다.

---

## 디자인 시스템 — Anchors 톤앤매너

> 출처: https://anchors-biz.com 분석 결과

### 컬러 팔레트

| 역할 | 색상 | 헥스 코드 |
|------|------|-----------|
| Primary (주조색) | 로열 블루 | `#1F0AAD` |
| Accent (포인트) | 시안 블루 | `#00BEFD` |
| CTA / 링크 | 애플 블루 | `#007AFF` |
| Background | 딥 블랙 | `#111111` |
| Surface | 다크 그레이 | `#0d0d0d` |
| Text Primary | 화이트 | `#FFFFFF` |
| Text Secondary | 미디엄 그레이 | `#BBBBBB` |
| Text Muted | 다크 그레이 | `#555555` |
| Danger | 레드 | `#FF6B6B` |

### 타이포그래피

- **폰트**: `Helvetica Neue`, sans-serif
- **헤딩**: `font-weight: 700`, `letter-spacing: -0.5px` — Bold하고 타이트한 느낌
- **본문**: `font-size: 14px`, `line-height: 1.75`
- **레이블/태그**: `font-size: 11–12px`, `font-weight: 500–600`

### UI 원칙

- 딥 다크 배경 기반의 다크 테마 전용 UI
- 강한 컬러 대비 (Primary Blue ↔ Cyan Accent)
- 여백을 넉넉하게, 텍스트 중심의 레이아웃
- 브랜드 포인트: 좌측 상단 로고 + Cyan dot 조합
- 호버 및 포커스 시 `#1F0AAD` → `#00BEFD` 계열로 전환

### 파일 상단 CHANGELOG 블록 (HTML 주석)

```
<!--
  CHANGELOG
  [YYYY-MM-DD HH:MM | vX.X.X] 변경 요약
    - 세부 변경 사항 1
    - 세부 변경 사항 2
-->
```

### 코드 내 인라인 주석

변경된 코드 블록 바로 위에 인라인 주석을 추가합니다.

```js
// [2026-04-21 14:30 | v1.2.0] 마크다운 렌더링 추가
function renderMarkdown(text) { ... }
```

```css
/* [2026-04-21 15:00 | v1.2.0] 라이트 테마 변수 추가 */
:root[data-theme="light"] { ... }
```
